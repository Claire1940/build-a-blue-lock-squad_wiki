<Section icon={Map} title="基礎：エリート選手獲得のためのリセマラ">

2026年の上位リーグで戦える**ブルーロックのチームを編成**するには、最初の試合をプレイする前から準備が始まります。リセマラは、序盤に貴重な通貨を消費することなく競争上の優位性を確保するための最も効率的な方法です。現在の2026年のメタでは、不十分なロースターで開始すると、ワールドチャンピオンロードの進行が大幅に遅れてしまいます。

ゲームを開始すると、通常2回のマルチ召喚を行うのに十分な通貨が付与されます。目的は、これらのガチャから少なくとも1体のトップティア星3キャラクターを確保することです。さらに、ゲーム内では星3確定チケットも提供されます。これらを組み合わせることで、チームの屋台骨となる星3ユニットを最低2体揃える必要があります。

| リセマラティア | 目標結果 | 推奨度 |
| :--- | :--- | :--- |
| **エリートスタート** | 星3キャラクター 4体以上 | 強く推奨 |
| **強力なスタート** | 星3キャラクター 3体 | ほとんどのプレイヤーに最適 |
| **標準的なスタート** | 星3キャラクター 2体 | 最低許容ライン |
| **不十分なスタート** | 星3キャラクター 1体 | リセマラを検討 |

<Callout type="warning" title="リセマラの重要ノート">
星3ユニット1体だけで妥協しないでください。中盤の難易度急上昇に対応するには、多様なロースターが必要です。最初のマルチ召喚で少なくとも2体のユニークな星3選手を引けなかった場合は、後でジェムを稼ぐよりも、データをリセットしてやり直す方が早いです。
</Callout>

<Section icon={Target} title="トレーニングサイクル：GランクからSSランクへ">

**ブルーロックのチームを編成**しようとする初心者が最も混乱する点の一つは、新しく入手したキャラクターをすぐにチームに加えられない理由です。このゲームでは、キャラクターはトレーニングセッションを受けるまで「Gランク」状態でロックされています。トレーニングとは、キャラクターのステータスを定義し、実際の試合で使用するためのバージョンを「確定」させるプロセスです。

**ビデオハイライト:**
- 24ターンのトレーニングシステムとステータス配分の詳細解説。
- 高ランクで終了するためのトレーニングマッチの進め方。
- キャラクターレベルとトレーニングランクの違いについての説明。

<YouTubeEmbed videoId="xuKpcrILeyE" title="[初心者ガイド] 最良の開始方法と進行手順" />

トレーニングプロセスは24ターンのカウントダウン形式で行われます。各アクションで1ターンを消費し、スピード、パワー、テクニックなどの特定のステータスが上昇します。目標は、キャラクターの戦力（BP）を最大化しながらターンをゼロにすることです。

<Steps>
  <Step num="1" title="トレーニング対象を選択">
    有効化したい星3キャラクターを選択します。セッションを途中で終了するとステータス上昇が不十分になる可能性があるため、開始前に十分なスタミナがあることを確認してください。
  </Step>
  <Step num="2" title="24ターンのグリッドを進める">
    トレーニングオプションをクリックして、ターンカウンターを24から0へ減らしていきます。最高のステータス倍率を得るために、「Extreme」または「Best」評価のトレーニングセッションに集中しましょう。
  </Step>
  <Step num="3" title="トレーニングマッチに勝利する">
    定期的に試合が発生します。これらに勝利することは非常に重要で、大量のBPブーストが得られ、1回のセッションでGランクからD、C、あるいはBランクまで一気にジャンプできるかどうかが決まります。
  </Step>
  <Step num="4" title="確定して編成する">
    ターンがゼロになると、キャラクターのランクが確定します（例：D+ランク）。これで、その「トレーニング済みバージョン」のキャラクターを正式にチームに加えることができます。
  </Step>
</Steps>

<Callout type="tip" title="ステータスバランスのコツ">
スピードに特化したくなりますが、初心者にはバランスの取れたビルドの方が適していることが多いです。高レベルの試合で当たり負けしないよう、筋力トレーニングや戦術ドリルで「Good」や「Extreme」評価を狙いましょう。
</Callout>

</Section>

<Section icon={Shield} title="高度なチーム最適化戦術">

初心者段階を抜けると、単に星3キャラクターを持っているだけでは不十分であることに気づくでしょう。トレーニングフェーズで使用する「サポート」キャラクターを最適化する必要があります。これらのサポートユニットは、標準的なDランクプレイヤーをSSランクの怪物に変えることができるパッシブボーナスを提供します。

| サポートタイプ | 恩恵 | 最適な対象 |
| :--- | :--- | :--- |
| **スピードブースター** | スピード獲得量が20%増加 | ウインガー、ストライカー |
| **戦術エキスパート** | テクニック獲得量が増加 | プレイメイカー、MF |
| **フィジカルトレーナー** | スタミナ消費を軽減 | ディフェンダー |

<Callout type="warning" title="よくある間違いを避ける">
多くのプレイヤーが、トレーニング成功後にチームを更新し忘れます。キャラクターをトレーニングして高いランク（例：DからBへ）を達成した場合は、手動でチーム編成メニューに行き、古いDランクバージョンを新しいBランクバージョンに入れ替える必要があります。
</Callout>

| ランク | BP範囲 | 達成難易度 |
| :--- | :--- | :--- |
| **G ～ E** | 0 - 5,000 | 自動 / 非常に簡単 |
| **D ～ C** | 5,001 - 15,000 | 基本的なトレーニング |
| **B ～ A** | 15,001 - 30,000 | サポートユニットが必要 |
| **S ～ SS** | 30,001+ | 専門的なステータス管理 |

</Section>
