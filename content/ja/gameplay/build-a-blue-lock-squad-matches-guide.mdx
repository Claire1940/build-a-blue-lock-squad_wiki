<Section icon={Target} title="フォーメーションとポジション連携の最適化">

フォーメーションは戦略の根幹です。よくある間違いは、チームの平均を上げるためだけに、高OVRの選手を間違ったポジションに配置することです。これにより、選手間の青い線で視覚化されるケミストリーリンクが切れてしまいます。これらの線が黄色になったり消えたりすると、個々の能力に関わらず、試合中のチームパフォーマンスは著しく低下します。

現在の2026年メタで最も効果的なフォーメーションは以下の通りです。エリートウィングがいるか、中央突破を好むかによって適切なものを選びましょう。

| フォーメーション | 最適なスタイル | 主な弱点 |
| :--- | :--- | :--- |
| **4-3-3** | サイド攻撃とクロス | カウンターに弱い |
| **3-4-3** | 最大限の攻撃プレッシャー | エリートなセンターバックが必要 |
| **4-4-2** | 守備の安定性 | 独創的なパスに欠けることがある |
| **4-5-1** | 中盤の支配 | ストライカーが孤立しやすい |

<Callout type="warning" title="ケミストリー注意報">
潔世一のような選手をSTからCMに移すのはパス能力を活かすのに役立つように見えますが、彼にCMのセカンダリポジションがない場合、中盤全体のケミストリーが低下し、ポゼッションを失う原因になります。
</Callout>

**最大ケミストリーのためのポジション優先順位:**
- **ゴールキーパー (GK):** 1名
- **ディフェンダー (LB, RB, CB):** 4名
- **ミッドフィルダー (LM, RM, CM):** 3名
- **フォワード (LW, RW, ST):** 3名

一部のレア選手は「フレックスポジション」を持っており、2つまたは3つの異なるスポットで100%のケミストリーを維持できます。これらの選手はチームの「ジョーカー」であり、リロールの際に優先すべきです。

<Section icon={Sparkles} title="選手の属性とエゴイストの特性">

この**試合ガイド**では、選手のパフォーマンスを6つの主要属性に分類しています。OVRは一般的な指標ですが、ステータスの具体的な配分がシミュレーション中の選手の動きを決定します。例えば、スピードが99でもシュートが70のストライカーは、ディフェンダーを追い越すことはできても、得点に失敗することが多いです。

| 属性 | 主な機能 | 理想的なポジション |
| :--- | :--- | :--- |
| **スピード** | 1秒あたりの移動距離 | LW, RW, LB, RB |
| **シュート** | シュートの精度と威力 | ST, LW, RW |
| **パス** | ロングパスとショートパスの成功率 | CM, LM, RM |
| **ドリブル** | ディフェンダーを抜く能力 | LM, RM, ST |
| **守備** | インターセプトとタックルの技術 | CB, LB, RB |
| **フィジカル** | ヘディングの競り合いとキープ力 | CB, ST |

</Callout>

<Section icon={Swords} title="ステップバイステップの試合準備">

高レベルの試合に備えるには、単に「プレイ」ボタンを押すだけでは不十分です。チームが休息できているか、戦術が設定されているか、控え選手が疲労したスタメンの代わりを務める準備ができているかを確認する必要があります。ワールドカップモードでの勝率を最大化するために、以下の手順に従ってください。

<Steps>
  <Step num="1" title="チーム確認">
    全選手のポジションを確認してください。全ての接続ラインが青であることを確認します。黄色のラインがある場合は、例えOVRが少し低くても、その役割に適した控え選手と交代させてください。
  </Step>
  <Step num="2" title="戦術調整">
    チームのメンタリティを設定します。格上の相手には「カウンター」モードを使用し、格下のチームには「ハイプレス」を使用して相手陣内でターンオーバーを誘発しましょう。
  </Step>
  <Step num="3" title="スキル割り当て">
    キャプテンに最も影響力のある「オーラ」スキルを装備させます。このスキルはチーム全体に影響を与え、試合の最終局面で勝敗を分ける要因になります。
  </Step>
  <Step num="4" title="スタミナチェック">
    選手のスタミナが100%であることを確認してください。スタミナが80%未満の選手は、全フィジカルステータスに10%のペナルティを受けた状態で試合を開始します。
  </Step>
</Steps>

ワールドカップのベスト32の試合は、準決勝よりもかなり簡単です。序盤のラウンドを利用して、後半のステージのために「フロー」メーターを貯めておきましょう。
</Callout>
